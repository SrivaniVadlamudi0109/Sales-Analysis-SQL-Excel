# Sales Analysis Dashboard using SQL & Excel

## Project Overview
This project analyzes retail sales data using MySQL and Microsoft Excel. SQL queries were used to extract business insights, and an interactive Excel dashboard was created using Pivot Tables and Pivot Charts.

## Tools Used
- MySQL
- Microsoft Excel

## Key Analysis
- Sales by Category
- Sales by Region
- Sales by Segment
- Top 10 Customers
- Top 10 Products

## Skills
SQL | Excel | Pivot Tables | Pivot Charts | Data Analysis | Data Visualization
